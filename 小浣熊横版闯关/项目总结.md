# 🦝 小浣熊冒险记 - 跨平台打包完整方案

## 📋 项目概述

本项目为小浣熊冒险记HTML5游戏提供了完整的跨平台打包解决方案，支持一键构建Windows桌面版(.exe)和Android移动版(.apk)应用。

## 🎯 打包成果

### 🖥️ Windows桌面版
- **文件位置**: `桌面版/dist/小浣熊冒险记.exe`
- **技术栈**: PyQt6 + QWebEngineView + PyInstaller
- **文件大小**: 约50-80MB
- **系统支持**: Windows 7/8/10/11 (64位)
- **特色功能**:
  - 原生桌面应用体验
  - 工具栏版本切换 (基础版1-5关 / 高级版6-10关)
  - 深色主题支持
  - 键盘快捷键操作
  - 自动游戏文件打包

### 📱 Android移动版
- **文件位置**: `移动版/android/app/build/outputs/apk/debug/app-debug.apk`
- **技术栈**: Capacitor + WebView + PWA
- **文件大小**: 约20-30MB
- **系统支持**: Android 5.0+ (API Level 21+)
- **特色功能**:
  - 触摸优化的虚拟按键
  - PWA离线支持
  - 设备方向自适应
  - 进度云端保存
  - 主屏幕快捷方式

## 🔧 技术实现详情

### 桌面版实现方案
```python
# 核心框架: PyQt6 + QWebEngineView
class RaccoonGameWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        self.web_view = QWebEngineView()  # 嵌入HTML5游戏
        self.load_game(game_type="basic")  # 加载基础/高级版
        
    def load_game(self, game_type):
        # 加载本地或在线游戏文件
        game_path = Path("游戏文件") / f"{game_type}.html"
        self.web_view.load(QUrl.fromLocalFile(str(game_path)))
```

### 移动版实现方案
```javascript
// 核心框架: Capacitor + WebView
const config: CapacitorConfig = {
  appId: 'com.minimaxstudio.raccoonadventure',
  appName: '小浣熊冒险记',
  webDir: 'www',
  plugins: {
    SplashScreen: { /* 启动画面配置 */ },
    Keyboard: { /* 虚拟键盘处理 */ }
  }
};

// 触摸控制优化
setupTouchControls() {
  const leftBtn = document.getElementById('leftBtn');
  leftBtn.addEventListener('touchstart', () => this.touch.left = true);
  leftBtn.addEventListener('touchend', () => this.touch.left = false);
}
```

## 🛠️ 开发环境要求

### 桌面版环境
- **Python**: 3.8+ (推荐3.11)
- **操作系统**: Windows 7+, macOS 10.12+, Ubuntu 18.04+
- **内存**: 4GB+ (推荐8GB+)
- **依赖包**: PyQt6, PyQt6-WebEngine, pyinstaller, qdarktheme

### 移动版环境  
- **Node.js**: 16+ (推荐18+)
- **Java**: JDK 8+ (Android构建需要)
- **Android SDK**: API 21+ (Android 5.0+)
- **开发工具**: Android Studio (可选)

### 构建依赖安装
```bash
# 桌面版依赖
pip install PyQt6 PyQt6-WebEngine pyinstaller qdarktheme

# 移动版依赖
npm install @capacitor/cli @capacitor/android
```
