# 🎮 完整控制指南

## 🖥️ 桌面端控制

### 键盘控制
| 按键 | 功能 | 说明 |
|------|------|------|
| `←` `→` | 左右移动 | 控制小浣熊左右移动 |
| `↑` `↓` | 上下移动 | 用于特定关卡的上下移动 |
| `空格键` | 跳跃 | 小浣熊跳跃动作 |
| `Enter` | 确认/动作 | 执行确认或特殊动作 |
| `ESC` | 返回菜单 | 从游戏返回到关卡选择 |
| `R` | 重新开始 | 重新开始当前关卡 |

## 📱 移动端控制

### 触屏虚拟按键
**方向控制区** (屏幕左下角)：
- `↑` 上方向键
- `←` 左方向键  
- `→` 右方向键
- `↓` 下方向键

**动作按键区** (屏幕右下角)：
- `跳` 跳跃按钮
- `A` 动作/确认按钮

## 🔧 故障排除

### 常见问题
1. **手柄无响应**
   - 检查USB连接或蓝牙配对
   - 尝试重新连接手柄
   - 确认浏览器支持Gamepad API

2. **触屏无反应**
   - 确认浏览器允许触屏权限
   - 检查屏幕旋转锁定
   - 尝试刷新页面

3. **键盘控制失效**
   - 检查页面焦点是否在游戏区域
   - 确认没有其他应用占用键盘
   - 尝试点击游戏区域重新获得焦点
