## 📋 本地构建步骤

### 1. 准备工作
```bash
# 克隆或下载项目文件
git clone [项目地址]
cd 小浣熊冒险记

# 安装桌面版依赖
cd 桌面版
pip install -r requirements.txt

# 安装移动版依赖
cd ../移动版
npm install
```

### 2. 桌面版构建
```bash
# Windows用户
build_all.bat

# macOS/Linux用户
bash build_all.sh
```

### 3. 移动版构建 (需安装Java和Android)
```bash
# 检查环境
python test_build.py

# 构建Android APK
cd 移动版
bash build_android.sh  # macOS/Linux
build_android.bat      # Windows
```

## 🔧 故障排除

### 桌面版问题
- **PyQt6错误**: 确保安装完整的Qt依赖
- **权限问题**: Linux/macOS下添加执行权限 `chmod +x 小浣熊冒险记`
- **依赖缺失**: 运行 `pip install -r requirements.txt`

### 移动版问题
- **Java错误**: 安装JDK 11+并配置JAVA_HOME
- **Android SDK错误**: 通过Android Studio安装完整SDK
- **Gradle错误**: 确保网络连接正常以下载依赖
