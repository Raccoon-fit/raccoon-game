# 小浣熊冒险记 - 游戏说明

## 🎮 游戏简介
一个温馨可爱的关卡制通关游戏，主角是一只憨态可掬的小浣熊。每一关都有独特的挑战和惊喜，越到后面越考验思维能力。

## 🎯 第1关玩法
**目标**：帮助小浣熊从趴着状态起来后，操纵它移动并跳跃到右边的树洞中。

## 🎮 完整控制支持

### 🖥️ 桌面端控制
- **键盘**：←→ 移动，空格键 跳跃，ESC 返回菜单，R 重新开始
- **手柄**：支持 Xbox、PlayStation、Nintendo 所有主流控制器
- **摇杆**：左摇杆移动，肩键跳跃，D-pad 方向控制

### 📱 移动端控制  
- **触屏**：专业设计的虚拟按键布局
- **方向键**：屏幕左下角的十字键控制
- **动作键**：屏幕右下角的跳跃和确认按钮
- **自适应**：根据屏幕尺寸自动调整布局

### 🎯 智能控制检测
- **自动识别**：自动检测并适配可用的控制设备
- **状态显示**：实时显示当前使用的控制器类型
- **无缝切换**：可在不同控制方式间自由切换

## 🎨 游戏特色
- **🦝 真实浣熊**：标志性眼罩、黑白花纹、长条纹尾巴
- **🎬 完整动画**：起身、待机、走路、跳跃的流畅动画
- **🌱 自然场景**：柔和摆动的草地、渐变天空、动态云朵
- **📱 跨平台**：完美支持桌面和移动端
- **🎮 全控制**：键盘 + 手柄 + 触屏全覆盖
- **♿ 无障碍**：高对比度、大按键、清晰指示

## 📱 设备支持

### 桌面端
- **浏览器**：Chrome、Firefox、Safari、Edge
- **键盘**：所有标准USB/蓝牙键盘
- **手柄**：Xbox Series X/S、Xbox One、PS5、PS4、Nintendo Pro、Joy-Con

### 移动端
- **iOS**：iPhone 6s+、iPad Air 2+
- **Android**：Android 7.0+ 设备
- **平板**：iPad、安卓平板横屏模式

### 客厅娱乐
- **智能电视**：支持浏览器的智能电视
- **游戏主机**：通过浏览器访问
- **投影仪**：连接电脑或手机

## 🚀 快速开始
1. 用浏览器打开 `index.html`
2. 选择"第1关"开始游戏
3. 等待小浣熊起身完成（3秒）
4. 使用任意支持的控制方式：
   - **键盘**：方向键移动 + 空格跳跃
   - **手柄**：摇杆移动 + 肩键跳跃
   - **触屏**：虚拟按键控制
5. 移动到发光树洞通关！
