# 小浣熊冒险记 - 跨平台打包方案

本项目提供完整的跨平台打包解决方案，将HTML5游戏包装成桌面版和移动版应用。

## 📦 打包方案

### 🖥️ Windows桌面版 (exe程序)
- **框架**: PyQt6 + QWebEngineView
- **特点**: 轻量级、跨平台、原生体验
- **文件**: 桌面版/ 目录

### 📱 Android移动版 (apk程序)
- **框架**: Capacitor + WebView
- **特点**: PWA支持、触摸优化、离线运行
- **文件**: 移动版/ 目录

## 🚀 快速开始

### Windows桌面版构建

1. **环境准备**
   ```bash
   # 确保已安装Python 3.8+
   python --version
   
   # 安装依赖
   pip install PyQt6 PyQt6-WebEngine qdarktheme pyinstaller
   ```

2. **运行构建脚本**
   ```bash
   # Windows
   cd 桌面版
   build_windows.bat
   
   # Linux/macOS
   cd 桌面版
   chmod +x build_windows.sh
   ./build_windows.sh
   ```

3. **输出结果**
   - `dist/小浣熊冒险记.exe` - 可执行文件
   - `小浣熊冒险记_v1.0_桌面版.zip` - 完整发布包

### Android移动版构建

1. **环境准备**
   ```bash
   # 确保已安装Node.js 16+
   node --version
   npm --version
   
   # 安装Android Studio和SDK
   # 设置 ANDROID_HOME 环境变量
   ```

2. **运行构建脚本**
   ```bash
   # Windows
   cd 移动版
   build_android.bat
   
   # Linux/macOS
   cd 移动版
   chmod +x build_android.sh
   ./build_android.sh
   ```

3. **输出结果**
   - `android/app/build/outputs/apk/debug/app-debug.apk` - APK文件
   - `发布包/Android/小浣熊冒险记_v1.0.apk` - 打包的APK

## 🔧 技术详情

### 桌面版技术栈

- **GUI框架**: PyQt6
- **Web引擎**: QWebEngineView (基于Chromium)
- **打包工具**: PyInstaller
- **主题**: QDarkTheme (可选)
- **支持**: Windows, macOS, Linux

### 移动版技术栈

- **混合框架**: Capacitor
- **Web引擎**: 系统WebView
- **打包工具**: Gradle + Android SDK
- **PWA支持**: Service Worker + Web App Manifest
- **支持**: Android 5.0+

## 🔐 签名和发布

### 桌面版签名
- 使用PyInstaller生成的exe文件
- 可使用代码签名证书进行签名
- Windows: `signtool.exe`
- macOS: `codesign`

### Android签名
```bash
# 生成发布密钥
keytool -genkey -v -keystore release-key.keystore -alias my-app

# 配置gradle.properties
RELEASE_STORE_FILE=../release-key.keystore
RELEASE_STORE_PASSWORD=your_password
RELEASE_KEY_ALIAS=my-app
RELEASE_KEY_PASSWORD=your_password

# 构建发布版APK
cd android
./gradlew assembleRelease
```

## 🐛 故障排除

### 常见问题

1. **Python依赖安装失败**
   ```bash
   pip install --upgrade pip
   pip install PyQt6 PyQt6-WebEngine
   ```

2. **Android构建失败**
   - 检查Android SDK安装
   - 设置ANDROID_HOME环境变量
   - 更新Gradle版本

3. **应用无法启动**
   - 检查系统兼容性
   - 查看错误日志
   - 尝试以管理员权限运行

4. **WebView显示异常**
   - 检查网络连接
   - 确认文件路径正确
   - 验证HTML/CSS语法
